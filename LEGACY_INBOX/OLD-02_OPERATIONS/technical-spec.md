# DGA Technical Specification

## System Architecture
- **Frontend:** Google Workspace (Forms, Sheets, Docs)
- **Automation:** n8n workflows
- **Configuration:** GitHub repository
- **AI Integration:** ChatGPT, Perplexity, Gemini APIs

## Core Workflows
1. Subscriber Onboarding
2. Territory Assignment  
3. GEO/AIO Visibility Audits
4. Content Plan Generation
5. 30-Day Recurring Cycles

## Data Models
- Territories: 78 Ontario regions with FSA mapping
- Niches: 11 REA specialties, 8 MP specialties
- Subscriptions: Playbook vs Solo streams
