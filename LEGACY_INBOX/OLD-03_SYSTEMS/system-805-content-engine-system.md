# System-805 – Content Engine System

*A core system within the Digital Growth Accelerator (DGA) Platform that converts strategy, voice, and GEO/AI insights into structured, multi-format content outputs aligned with the Omni-Channel Growth Framework (OCGF).*

---

## Purpose

System-805 operationalizes content creation across all channels using:

- The client’s Brand Voice Blueprint (System-802)
- GEO/AI audit findings (System-801)
- Niche & Persona intelligence (Niche-2xx / Persona-1xx)
- OCGF distribution rules
- DGA Standards (601–607)

This system allows any assistant (human or AI) to produce consistent, accurate, on-brand content in formats that support visibility, authority, and discoverability.

---

## Category Code

`SYSTEM`

## Prompt Type

`FUNCTION`

---

## Description

System-805 produces structured content outputs such as:

- Long-form posts  
- Short-form posts  
- Carousel scripts  
- Email drafts  
- Website copy  
- Google Business Profile posts  
- Video/Reel scripts  
- Niche-aligned and persona-aligned content  
- OCGF derivative content  
- Content kits and weekly packs  

It ensures all content:

1. Uses the client’s authentic Brand Voice  
2. Aligns with their niche + persona classification  
3. Uses anchor topics derived from the OCGF  
4. Improves GEO/AI clarity for future search visibility  
5. Follows the DGA Standards and formatting rules  
6. Is implementation-ready for client publishing  

This system powers all Content-3xx prompts.

---

# Inputs Expected

## 1. Client Voice Inputs
- Brand Voice Blueprint (BVB)
- Key phrases, tone rules, and examples
- Voice constraints and “never use” items

## 2. Niche & Persona Inputs
- Primary niche  
- Supporting niche(s)  
- Persona classification  
- Market focus  
- Audience pain points  

## 3. Content Strategy Inputs
- Topic or anchor theme  
- OCGF distribution plan  
- Content type (long-form, short-form, carousel, script, email, etc.)  
- Use case (awareness, teaching, credibility, call-to-action)  

## 4. Platform Context
- Instagram, Facebook, LinkedIn  
- YouTube Shorts / Reels  
- Google Business Profile  
- Website or landing page  
- Email newsletter  

## 5. Constraints
- Canada-only mortgage context  
- No emojis unless explicitly requested  
- Tone must follow BVB  
- Use client-friendly, clear language  
- Must follow DGA Standards 601→607  

---

# Outputs

## 1. Primary Content Asset
A full, polished draft of the requested content type using:

- Brand Voice  
- Persona/Niche alignment  
- GEO/AI clarity rules  
- OCGF structure  
- System-level formatting standards  

Examples:
- A 700-word educational LinkedIn post  
- A 45-second Reel script  
- A multi-slide carousel outline  
- A GBP post (100–300 characters)  
- An email announcement  

## 2. OCGF Derivative Set (Optional)
If requested, System-805 automatically generates:

- 1 long-form  
- 1 short-form  
- 1 carousel  
- 1 reel/short script  
- 1 email  
- 1 GBP post  

All from the same anchor topic.

## 3. SEO / GEO / AI Alignment Notes
Short notes explaining:

- Entity clarity  
- Search visibility improvements  
- AI recommendation readiness  
- Narrative alignment  

## 4. Suggested Hashtags / Metadata
Only if requested — otherwise omitted.

## 5. Implementation Notes
- Posting cadence  
- Suggested visuals  
- Calls-to-action  
- Where this fits in the broader OCGF
